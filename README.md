# Анализ временных рядов - Курсы INR/RUB

Приложение для анализа курса индийской рупии (INR) к российскому рублю (RUB).

## Функционал
- Загрузка данных с ЦБ РФ
- Разделение данных (по годам, неделям, X/Y)
- Статистический анализ
- Визуализация данных
- GUI интерфейс

## Установка и запуск

### Способ 1: EXE-файл
1. Запустите `CurrencyAnalysis.exe` из папки `dist/`

### Способ 2: Исходный код
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите: `python main.py`

### Тестирование
Запуск тестов: `python -m pytest tests/ -v`

## Используемые библиотеки
- PySide6 - GUI
- pandas, numpy - анализ данных
- matplotlib - визуализация



# Руководство пользователя

## Быстрый старт
1. Запустите приложение
2. Выберите папку с данными
3. Используйте вкладки для доступа к функционалу

## Основные функции
- **Основной функционал**: загрузка данных, поиск по дате
- **Расширенный анализ**: статистика, фильтрация, графики

## Ограничения
- Данные доступны с 2016 года
- Требуется интернет для загрузки данных
